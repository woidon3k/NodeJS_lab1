# Market Management System (AntonRetail) 

Веб-застосунок для управління складом роздрібного магазину. 

## Функціонал
- Адаптивний інтерфейс (ПК/Планшет/Мобільні).
- Моніторинг термінів придатності (автоматичне підсвічування статусів).
- Контроль товарних залишків.
- Адаптивне навігаційне меню (бургер-меню).

## Технології
- HTML5 
- CSS3 (Flexbox, Grid, Media Queries).
- JavaScript (DOM Manipulation, Mock Data).

## Як запустити
1. Склонуйте репозиторій: `git clone https://github.com/woidon3k/NodeJS_lab1`
2. Відкрийте файл `index.html` у вашому браузері.
